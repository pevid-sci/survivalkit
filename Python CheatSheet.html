🔢 Operadores, Lógica e Controle

Operadores Essenciais: == (Igual), != (Diferente), > (Maior), < (Menor), // (Divisão Inteira), % (Resto), ** (Potência).
Lógica: and, or, not, in (está contido), is (identidade de memória).
if/elif/else (Python Base): Estrutura de decisão lógica
if p_valor < 0.05: status = "Significativo" elif p_valor < 0.1: status = "Tendência" else: status = "NS" # define rótulo baseado na evidência
list comprehension (Python Fluente): Criação concisa de listas filtradas
[x.upper() for x in colunas if "ID" not in x] # formata nomes de colunas
match/case (Python 3.10+): Pattern matching estrutural para múltiplas condições
match resultado: case {"status": "ok", "valor": v}: print(v) case {"status": "erro"}: print("Falha") # desempacota e testa simultaneamente

📦 Estruturas de Dados e Python Fluente

namedtuple (collections): Tuplas com nomes de campo (objetos leves)
Amostra = namedtuple('Amostra', ['id', 'grupo', 'desfecho']) obs = Amostra(1, 'A', 0) # acessível via obs.grupo
zip / enumerate (Python Base): Iteração paralela e com índices
for i, (par1, par2) in enumerate(zip(lista1, lista2)): print(f"Par {i}: {par1}, {par2}") # une listas e conta iterações
__getitem__ / __len__ (Métodos Especiais): Tornam objetos "Pythonicos" (fatiáveis e mensuráveis)
def __len__(self): return len(self._dados) # permite usar len(objeto)

🏗️ Programação Orientada a Objetos (Classes)

Classe: Um molde (projeto). Objeto: Uma instância do molde.
Atributo: Variável interna (característica). Método: Função interna (comportamento).
class / __init__ (POO): Define a estrutura e inicializa o estado do objeto
class EstudoClinico: def __init__(self, n, fase): self.n = n # Atributo de instância self.fase = fase def reportar(self): return f"Fase {self.fase} com N={self.n}" # Método
@property (Decorador): Transforma um método em um atributo de leitura (calculado)
@property def taxa_sucesso(self): return self.eventos / self.n # acessado como objeto.taxa_sucesso

📊 Manipulação e Análise (Pandas & NumPy)

read_csv / to_excel (pandas): I/O de dados de alta performance
df = pd.read_csv("data.csv", sep=";", encoding="latin-1") # importa base médica
loc / iloc (pandas): Seleção por rótulo ou por posição
df.loc[df['idade'] > 60, ['id', 'status']] # filtra linhas e colunas
groupby / agg (pandas): Agregamento e sumário estatístico por grupo
df.groupby('braço').agg({'PAS': ['mean', 'std'], 'evento': 'sum'}) # tabela descritiva
melt / pivot_table (pandas): Reshape de dados (Largo vs. Longo)
df_longo = pd.melt(df, id_vars=['id'], value_vars=['t0', 't1', 't2']) # prepara para Mixed Models

🔬 Bioestatística, Sobrevivência e Meta-Pesquisa

ttest_ind / f_oneway (scipy.stats): Testes de hipótese paramétricos
stats.ttest_ind(grupo_a, grupo_b, equal_var=False) # Teste t de Welch
logit / ols (statsmodels): Modelagem de regressão via fórmulas (estilo R)
smf.logit("obito ~ tratamento + idade + bmi", data=df).fit().summary() # calcula OR ajustada
KaplanMeierFitter (lifelines): Estima a função de sobrevivência
kmf.fit(duracao, event_observed=censura).plot_survival_function() # gera curva de KM
CoxPHFitter (lifelines): Regressão de riscos proporcionais de Cox
cph.fit(df, duration_col='tempo', event_col='evento') # extrai Hazard Ratio (HR)

🕸️ Inferência Causal e Modelagem Gráfica

CausalModel (dowhy): Framework para estimativa de efeitos causais
model = CausalModel(data=df, treatment='T', outcome='Y', common_causes=['W']) model.identify_effect() # encontra o caminho causal via backdoor criterion
DAG (causalgraphicalmodels): Definição e visualização de grafos acíclicos
dag = DAG(nodes=["X", "Y", "Z"], edges=[("X", "Y"), ("Z", "X"), ("Z", "Y")]) dag.draw() # visualiza estruturas de confusão e colisão

🌐 APIs, Web Scraping e Dados

requests.get / .json() (requests): Consumo de dados de APIs externas

soup = BeautifulSoup(html, 'html.parser') links = [a['href'] for a in soup.find_all('a', href=True)] # minera links de artigos

🛠️ Utilitários e Organização de Projeto

Path (pathlib): Manipulação de arquivos e caminhos agnóstica ao SO

re.search / re.sub (re): Expressões regulares para limpeza de texto
limpo = re.sub(r"\[\d+\]", "", texto_artigo) # remove citações estilo [1, 2]
logging.basicConfig (logging): Registro de logs para monitoramento de scripts
logging.info("Processamento da base iniciado") # gera rastro para depuração
